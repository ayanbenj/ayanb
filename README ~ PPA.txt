READ ME
PPA (Pokemon Personality App)
How does it work?
Input the answers for the questions in the boxes
After putting in each answer, click the CLICK ME! button
Then click the PERSONALIZE button

This widget/app is a work in progress if there are any errors, 
it is being worked on to its perfection. 

This is a personal project that I have wanted to do when I first was introduced to 
tkinter, and many many more widgets and personal projects that 
I want to create through GUI in Python. 

This widget is to receive information from the user pertaining their personality, 
favorite color, and favorite Pokemon type. 
After clicking the event buttons of CLICK ME and PERSONALIZE, 
there is to be an image of what Pokemon that can associated with the users 
given answers.
